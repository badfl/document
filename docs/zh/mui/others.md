## mui-content-padded

设置10px的外边距
**Mui.css （*v3.0.0*）部分源码：**
```css
.mui-content-padded
{
  margin: 10px;
}
```

## mui-clearfix

```css
/* 483 */
.mui-clearfix:before, .mui-clearfix:after
{
display: table;

content: ' ';
}
.mui-clearfix:after
{
clear: both;
}

```
